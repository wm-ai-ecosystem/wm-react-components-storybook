# Callback Events

<details open>
  <summary>Callback Events</summary>
    <div>
        | Event | Description |
        | --- | --- |
        | `onSubmit` | This event handler is called whenever a submit event is triggered. |
        | `onBeforeRender` | This event handler is called before rendering the form fields. |
        | `onSucess` | This event handler is called whenever a success event is triggered. |
        | `onError` | This event handler is called whenever an error event is triggered. |
    </div>
</details>
