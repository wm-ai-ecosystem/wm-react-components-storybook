# Callback Events

<details open>
  <summary>Panel Events</summary>
    <div>
| Event | Description |
|-------|-------------|
| `onLoad` | This event handler is called when the component is loaded. |
| `onClose` | This event handler is called when the component is closed using the close button |
| `onCollapse` | Triggered when the panel is collapsed |
| `onExpand` | Triggered when the panel is expanded |
| `onFullscreen` | Triggered when the panel enters full-screen mode |
| `onExitfullscreen` | Triggered when the panel exits full-screen mode |
| `onActionsclick` | Triggered when an action item in the menu is clicked |
| `onPropertyChange` | Triggered when any property of the panel is changed |
    </div>
</details>

<details >
  <summary>Mouse Events</summary>
    <div>
| Event | Description |
|-------|-------------|
| `onMouseEnter` | This event handler is called whenever the mouse enters the component. |
| `onMouseLeave` | This event handler is called whenever the mouse leaves the component. |
| `onMouseover` | Triggered when the mouse pointer moves over the component |
| `onMouseout` | Triggered when the mouse pointer moves out of the component |
 </div>
</details>
