# Callback Events

<details open>
  <summary>Callback Events</summary>
    <div>
        | Event | Description |
        | --- | --- |
        | `onSelect` | This event is triggered when a user selects one or more dates. |
        | `onBeforeRender` | This event handler is called before rendering the component. |
        | `onEventDrop` | This event handler is triggered when dragging of an event in calendar stops and the event has moved to a different day/time. |
        | `onEventResize` | This event handler is triggered when resizing of the event in calendar stops and the event has changed in duration. |
        | `onEventClick` | This event handler is triggered when an event is clicked in the calendar. |
        | `onEventRender` | This event handler is triggered when an event is rendered. |
        | `onViewRender` | This event handler is triggered when the calendar view is changed. |
        | `onDateClick` | This event handler is triggered when a date is clicked in calendar component. |
    </div>
</details>
