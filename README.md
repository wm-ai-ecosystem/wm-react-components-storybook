# Yalc Setup for @wavemaker/react-runtime

This README provides the steps for setting up and using Yalc for publishing and adding the `@wavemaker/react-runtime` package in your projects.

## Prerequisites

- Node.js and npm installed.
- Yalc installed globally. If not, you can install it with the following command:
  ```bash
  npm install -g yalc
  ```

## Steps to Publish the Package to Yalc

- Go to the directory of the project where @wavemaker/react-runtime is developed.
  ```bash
   yalc publish --private
  ```
- Verify the Package is Published
  cd /Users/your-username/.yalc/packages/@wavemaker/react-runtime
